Компонент "sendMail" (Отправка писем с сайта)

Версия 1.0


----------------------
Возможности:

Помимо старндартных ыозможностей любого другого аналогичного решения по отрпавке писем с сайта (проверка данных (как на стороне клиента через JS-библиотеку, так и на стороне сервера), переданных формой; непосредственно отправка письма), данный компонент предоставляет следующие возможности:

- Добавление собственных шаблонов e-mail писем. Все шаблоны лежат в отдельной директории компонента.
- Возможность предварительной настройки опций компонента:
    - Название сайта, с которого отправляется почта;
    - E-mail, используемый для отправки писем с сайта на административные адреса;
    - Административные адреса, на которые будут отправляться служебные письма;
    - От имени кого будет отправлено E-mail на административные адреса:
        - от имени отправителя;
        - от имени сайта.
    - Кому необходимо отвечать на письмо:
        - отвечать отправителю;
        - отвечать сайту.
    - Имя шаблона письма по умолчанию;
    - Тема письма по умолчанию;
    - Сообщение по умолчанию об успешном результате операции отправки письма;
    - Сообщение по умолчанию об ошибке операции отправки письма;
- Идентификация пользователя по тестам Тьюринга:
    - Тест №1 "Проверка содержимого скрытого поля";
    - Тест №2 "Проверка минимальной длины номера телефона".
- Вывод сообщений о результате операции отправки почты.


----------------------
Функции:

На стороне клиента (JavaScript):
- Деативация кнопок действий ("Отправить", "Отмена", "Сбросить") формы при загрузке страницы;
- Активация/Деактивация кнопок действий ("Отправить", "Отмена", "Сбросить") формы в зависимости от того, заполнены все обязательные поля формы или нет;
- Проверка данных формы на пустые значения;
- Проверка данных формы на допустимость и корректность значений полей формы;
- Идентификация пользователя по тестам Тьюринга:
    - Проверка значения скрытого системного поля;
    - Проверка минимально необходимой длины номера телефона.
- Отмена действия по умолчанию браузера на событие отправки формы в случае возникновения ошибок в данных, выявленных в решультате их проверки;

На стороне Сервера (PHP):
- Проверка данных формы на пустые значения;
- Проверка данных формы на допустимость и корректность значений полей формы:
    - Проверка существования e-mail;
- Идентификация пользователя по тестам Тьюринга:
    - Проверка значения скрытого системного поля;
    - Проверка минимально необходимой длины номера телефона.
    

----------------------
Примечания:

Данная версия не поддерживает асинхронную отправку данных формы серверному скрипту!
